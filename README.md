# keepassxc.org
Website sources for https://keepassxc.org.

This website use **Jekyll**.

## Usage

- Install NPM dependencies: `npm install`
- Install Jekyll dependencies from Gemfile: `bundle install`
- Run Jekyll dev server: `jekyll serve`
